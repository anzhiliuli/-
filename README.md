# 碧蓝档案轴计算器

一个用于碧蓝档案游戏的轴计算器工具，帮助玩家规划和计算角色技能释放的时间轴。

## 功能特性

- 🎯 **角色技能计算**：精确计算角色技能的释放时间
- 📊 **可视化界面**：直观的UI界面，易于操作
- 🎨 **主题切换**：支持亮色和暗色主题
- 💾 **数据持久化**：保存用户配置和计算结果
- 📱 **响应式设计**：适配不同屏幕尺寸

## 快速开始

### 直接使用

1. 下载或克隆项目到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 开始使用计算器功能

### 开发模式

1. 安装项目依赖（如果有）
   ```bash
   npm install
   ```

2. 启动本地服务器（可选）
   ```bash
   npm run serve
   ```

3. 在浏览器中访问 `http://localhost:8000`

## 使用说明

1. **选择角色**：从下拉菜单中选择需要计算的角色
2. **配置参数**：设置技能等级、装备属性等参数
3. **计算轴**：点击计算按钮生成技能释放时间轴
4. **查看结果**：在结果区域查看计算出的时间轴
5. **导出数据**：可以将计算结果导出为文本或其他格式

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式设计（使用CSS变量和模块化）
- **JavaScript (ES6+)**：功能实现
- **Bootstrap 5**：UI组件库
- **Font Awesome**：图标库
- **Chart.js**：数据可视化

## 项目维护

### 项目结构说明

- **CSS模块化**：将样式分为变量、工具类、主题和组件，便于维护
- **JavaScript模块化**：使用ES6模块系统，提高代码可维护性
- **主题系统**：支持亮色和暗色主题切换
- **响应式设计**：适配不同屏幕尺寸

### 核心模块

- `app-state.js`：应用状态管理
- `data-processor.js`：数据处理逻辑
- `ui-renderer.js`：UI渲染
- `event-listeners.js`：事件监听
- `character-presets.js`：角色预设数据
- `dropdown-menu.js`：下拉菜单组件
